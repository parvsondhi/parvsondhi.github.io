G.E.A.R - Gest-Ex Augmented Reality


A follow up to Gest Ex, G.E.A.R is a cost effective Wireless/Portable Multi Gesture Interactive environment. Consisting of WII camera mounted helmet and special IR LED gloves, the portable set up alllowed users to easily interact with 3D augmented objects in a virtual space wirelessly through Bluetooth. Research was done on designing an algorithm using stereoscopic triangulation to convert 2D coordinates into 3D coordinates by tracking 4 IR points using 2 wiimotes to provide a 6 DoF system.

GEST-Ex


As part of the Gesture-Expression(GEST-Ex) project, we worked on Image manipulation and other applications using Hand gesture Detection via web cameras. The project involved the optimization of gesture and hand movement detection using the Canny and Bounding box algorithm for edge and contour detection on the video obtained from the system camera. Proposed Gesture Control Interaction System For Surgeons in Operating Theatres to avoid interaction with non-sterile surfaces of the devices in use and to reduce the risk of infection.

CLASSIFY



Research project to study the effect of multiple varied training data on statistical methods for classification. The research included the study of Naive Bayes Classifier as a supervised learning method to predict the genre of text summaries extracted from the images of multiple textbooks.
The aim was to succesfully classify the novels/textbooks into major genres such as action, romance, etc and study the effect on the bayesian classifier based on different training sets.

iPAT


The Personal Software Process (PSP) is a structured software development process that is intended to help software engineers understand and improve their performance, by using a "disciplined, data-driven procedure". The PSP Automation and Analysis Tool was intended to allow the user to simplify the PSP creation and analysing process. We researched basic statistical approaches to help study the inputs of the students and provide a better analysis of the students progress across the course of a semester.

Comic Con


A complete entertainment website created to cater users interested in everything related to Games, Movies, TV. The project was an attempt to work with HTML, PHP, Javascript, Actionscript. Under this project, the main emphasis was on creating a Flash based application which allowed users to create their own comic strips using their beloved characters. Also as part of the was the implementation of an efficient web parser to browse through all the related information online and retrieve relevant information.
